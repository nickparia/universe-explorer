UNIVERSE EXPLORER

8K PHOTOREALISTIC · SETUP REQUIRED
STEP 01

DOWNLOAD TEXTURES

Go to Solar System Scope and download the free 8K texture pack (CC-BY 4.0):

solarsystemscope.com/textures →

Download each planet. Free account required. File size ~600MB total.

STEP 02

ARRANGE FILES

Create a textures/ folder next to this HTML file and place the downloaded .jpg/.png files inside:

textures/
  8k_sun.jpg
  8k_mercury.jpg
  8k_venus_surface.jpg
  8k_earth_daymap.jpg
  8k_earth_nightmap.jpg
  8k_earth_clouds.jpg

  2k_uranus.jpg
  2k_neptune.jpg
STEP 03

SERVE LOCALLY

Browsers block local file loading. Run a local server from the folder containing this HTML file:

python -m http.server 8080

Then open in your browser:

localhost:8080

Or use VS Code Live Server, Node's npx serve, etc.

Missing textures fall back to procedural generation — you can launch without any textures to preview.
